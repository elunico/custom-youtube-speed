Key Bindings for Custom Video Speed for YouTube™

Here you can set custom keys for performing actions added by the Custom Video Speed for YouTube™ extension

You may click the "Set Key" button next to the action you want to customize and then strike the key you wish to use for this action. Be sure to hit save at the bottom of the page for your changes to take effect

Note: Not all keys will work this way, such as escape, command, etc. Furthermore, any keys that conflict with YouTube's builtin controls such as 'l', 'k', 'j', and the arrow keys are very buggy and I would not suggest using these until a fix can be implemented.

Below you have some options for how to treat your custom shortcut keys. If you do not wish to altar the behavior of Custom YouTube Speed 5.1.0, do not change these settings

  • Prevent default - stops letters from appearing in text areas. Prevents default normal behavior of your keys
  • Stop Propogation - prevent additional events from firing

If you would like to use a troublesome key, you may manually enter the key JavaScript Event value for it. If you don't know what this means, you can skip reading this section, just use the button and the keys you want and watch out for the built-in YouTube shortcuts.

You can use this website to discover the key of striking any key on the keyboard. Be warned that there is no guarantee that keys manually entered using key will actually work but it is possible to try.
